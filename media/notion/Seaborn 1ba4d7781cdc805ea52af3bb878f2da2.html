그래프 종류

  • 예제 데이터

    data = pd.DataFrame(data={

  • 막대 그래프
    sns.barplot(data=data)
  • Heatmap

    Seaborn의 heatmap은 2차원 데이터의 값을 색상으로 표현하여 데이터의 패턴, 분포, 상관관계 등을 시각적으로 쉽게 파악할 수 있는 그래프입니다. 주로 상관관계 분석이나 행렬 데이터를 시각화할 때 많이 사용됩니다.

    Heatmap은 각 셀의 색상이 해당 셀의 값의 크기를 나타내며, 색상은 기본적으로 밝기나 색조를 통해 값의 크기(높거나 낮음)를 표현합니다.

    Heatmap을 사용하는 경우

    • 상관관계 행렬 시각화: 데이터의 여러 변수 간의 상관관계를 시각화하는 데 유용합니다. 상관계수가 높을수록 양의 상관관계(양수), 낮을수록 음의 상관관계(음수)를 가지며, heatmap을 통해 이 관계를 한눈에 확인할 수 있습니다.
    • 행렬 데이터 시각화: 숫자나 값으로 구성된 2차원 행렬의 패턴을 파악할 때 사용합니다.
    • 분포 및 패턴 탐색: 특정 데이터셋에서 값의 분포나 패턴을 쉽게 이해하고자 할 때 활용됩니다.
    sns.heatmap(data)
    • 데이터 값을 표시하려면 annot=True를 지정한다.
    • fmt(포맷) 인자는 d, .2f와 같이 지정할 수 있다.
    sns.heatmap(data, annot = True, fmt = ".2f")
    • 색상 및 선 스타일 설정
    sns.heatmap(data, annot = True, fmt = ".2f", cmap = "Blues", linewidth = 1, linecolor = "black")
