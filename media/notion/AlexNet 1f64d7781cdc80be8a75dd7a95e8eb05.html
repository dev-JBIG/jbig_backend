AlexNet

기존의 문제점

기존 이미지 인식 기법은 SIFT나 HOG 같은 수작업 특징 추출에 의존하였다. 그래서 복잡한 실제 환경에서 성능이 급격히 떨어졌고, LeNet 등 얕은(layer가 작은) CNN은 MNIST 같은 단순 데이터에만 적합했다. 또한sigmoid/tanh 활성화 함수 사용으로 기울기 소실이 일어나 깊은 네트워크 학습이 어려웠다.

  • 기울기 소실이란?

    기울기 소실(Vanishing Gradient)이란, 신경망을 학습할 때 역전파로 전달되는 기울기(gradient)가 레이어를 거듭할수록 점점 작아져서 ‘앞쪽’ 층(입력에 가까운 층)으로 가면 거의 0이 되어 버리는 현상

AlexNet

총 8개의 학습층으로 구성되었으며, 5개의 합성층(C1 ~ C5)와 3개의 완전연결층 (F1 ~ F3)로 구성되어있다.

주요 기능

  1. ReLU 활성화 함수
    • 기존 sigmoid 대신 쓰여, 학습 속도가 6배 이상 빨라진다.
  1. 드롭아웃(Dropout)
    • 완전 연결 층에서 일부 뉴런을 임시로 끄며 over-fitting을 방지한다.
  1. 데이터 증강(Data Augmetation)
    • 원본 이미지에 랜덤 크롭·뒤집기·색 변화 등을 적용해, 데이터를 추가로 만든다.
  1. Local Response Normailization (LRN(
    • 주변 채널과 경쟁하듯 응답값을 정규화해, 중요한 특징이 더 도드라지게 함
  1. 오버래핑 풀링(Overlapping Pooping)
    • 풀링 창이 살짝 겹치게(3×3 창, 스트라이드 2) 써서 정보 손실을 줄임

import torch
import torch.nn as nn

class AlexNet(nn.Module):
    def __init__(self, num_classes=1000):
        super(AlexNet, self).__init__()
        # 특징 추출부 (Feature Extractor)
        self.features = nn.Sequential(
            nn.Conv2d(3, 64, kernel_size=11, stride=4, padding=2),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=3, stride=2),
            nn.Conv2d(64, 192, kernel_size=5, padding=2),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=3, stride=2),
            nn.Conv2d(192, 384, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),
            nn.Conv2d(384, 256, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),
            nn.Conv2d(256, 256, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=3, stride=2),
        )
        # 분류기 (Classifier)
        self.classifier = nn.Sequential(
            nn.Dropout(),
            nn.Linear(256 * 6 * 6, 4096),
            nn.ReLU(inplace=True),
            nn.Dropout(),
            nn.Linear(4096, 4096),
            nn.ReLU(inplace=True),
            nn.Linear(4096, num_classes),
        )

    def forward(self, x):
        x = self.features(x)
        x = x.view(x.size(0), 256 * 6 * 6)  # Flatten
        x = self.classifier(x)
        return x

if __name__ == "__main__":
    # 모델 생성 및 구조 출력
    model = AlexNet(num_classes=1000)
    print(model)
    # 임의의 입력 데이터로 전방 패스 테스트
    input_tensor = torch.randn(1, 3, 224, 224)
    output = model(input_tensor)
    print(output.shape)  # torch.Size([1, 1000])
