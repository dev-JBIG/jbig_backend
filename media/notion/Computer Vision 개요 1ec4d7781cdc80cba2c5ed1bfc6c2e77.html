Computer Vision 개요

이미지 딥러닝은 이미지 데이터를 다양한 방식으로 분석하고 처리하여 유용한 정보를 추출하는 기술입니다. 이미지 딥러닝의 주요 종류로는 분류(Classification), 객체 탐지(Object Detection), 세그멘테이션(Segmentation) 등이 있으며, 각각의 개념을 이해하면 이미지 데이터를 분석하는 다양한 목적을 효과적으로 달성할 수 있습니다.

1. 이미지 분류 (Image Classification)

  • 개념: 이미지 분류는 주어진 이미지가 특정 클래스에 속하는지를 예측하는 문제입니다. 모델은 입력된 이미지가 무엇을 나타내는지(예: 고양이, 개, 자동차 등)를 분류합니다.
  • 예시: 고양이와 개가 포함된 데이터셋에서, 모델이 이미지를 보고 고양이인지 개인지 구분하도록 학습합니다.
  • 응용 분야: 사람 얼굴 인증, 품목 분류, X-ray 이미지 진단 등

2. 객체 탐지 (Object Detection)

  • 개념: 객체 탐지는 이미지 내의 특정 객체의 위치를 찾아내고, 그 객체가 어떤 종류인지 분류하는 문제입니다. Bounding Box라 불리는 사각형 틀로 객체의 위치를 표시합니다.
  • 예시: 사진 속 여러 동물의 위치를 탐지하여 각 사각형 박스에 어떤 동물인지 라벨링합니다.
  • 응용 분야: 자율 주행차의 보행자 탐지, CCTV 영상의 사람 탐지, 물체 위치 기반 로봇 제어 등
  • 주요 모델: YOLO(You Only Look Once), Faster R-CNN, SSD(Single Shot MultiBox Detector)

3. 이미지 세그멘테이션 (Image Segmentation)

이미지 세그멘테이션은 이미지의 모든 픽셀을 분류하는 작업으로, 객체의 경계까지 포함해 더 정확하게 객체 영역을 구분할 수 있습니다. 세그멘테이션은 크게 두 가지 종류로 나뉩니다.

(1) Semantic Segmentation

  • 개념: 이미지 속의 픽셀 하나하나를 특정 클래스로 분류하여 이미지 전체를 객체별로 구분합니다. 다만, 같은 객체라면 개별적인 구분은 하지 않습니다.
  • 예시: 도로 위 사진에서 도로, 차량, 보행자 등을 픽셀 단위로 구분하여 각 픽셀에 ‘차량’, ‘사람’ 등의 라벨을 붙입니다.
  • 응용 분야: 의료 이미지 분석, 자율 주행, 지리 정보 시스템 등

(2) Instance Segmentation

  • 개념: Semantic Segmentation과 유사하지만, 같은 클래스의 객체가 여러 개 있을 때 각각을 별도로 구분합니다.
  • 예시: 다수의 사람이 있는 이미지에서, 각 사람을 서로 다른 인스턴스로 분리하여 표시합니다.
  • 응용 분야: 다양한 개체들이 밀집된 이미지를 분석하는 분야(예: 군중 분석, 마이크로 구조 분석 등)
  • 주요 모델: Mask R-CNN, DeepLab

4. 이미지 생성 (Image Generation)

  • 개념: 이미지 생성은 학습한 데이터를 기반으로 새로운 이미지를 생성하는 작업입니다. 모델이 기존 이미지를 학습한 후, 전혀 새로운 이미지를 만들어냅니다.
  • 예시: 가상의 얼굴 생성, 애니메이션 캐릭터 생성, 저해상도 이미지를 고해상도로 복원
  • 응용 분야: 게임 그래픽 생성, 예술 창작, 의료 영상의 고해상도 변환 등
  • 주요 모델: GAN(Generative Adversarial Networks), VAE(Variational Autoencoder)

5. 이미지 캡셔닝 (Image Captioning)

  • 개념: 이미지 캡셔닝은 이미지를 보고 자연어로 설명을 생성하는 작업입니다. 이미지의 내용을 이해하여 간단한 설명을 제공합니다.
  • 예시: '한 남자가 자전거를 타고 있다', '두 마리의 고양이가 나무 아래에 있다'와 같이 이미지의 상황을 텍스트로 표현합니다.
  • 응용 분야: 사진 설명 생성, 시각장애인을 위한 텍스트 기반 설명, 동영상 자막 자동 생성 등
  • 주요 모델: CNN + RNN(Convolutional Neural Network + Recurrent Neural Network) 조합, Transformer 기반 모델

문제

Q: 이미지 분류(Image Classification)와 객체 탐지(Object Detection)의 주요 차이점은 무엇인가요?

  • 정답

    이미지 분류는 이미지 전체가 특정 클래스에 속하는지 예측하는 작업으로, 이미지를 하나의 카테고리로 분류합니다. 반면, 객체 탐지는 이미지 내에서 여러 개의 특정 객체를 찾아내어 각각의 위치(Bounding Box)와 클래스 정보를 제공합니다.
