스태킹

스태킹의 개념

  • 스태킹은 여러 모델의 예측값을 결합하여 최종 예측을 만드는 앙상블 기법입니다.
  • 기본 모델(개별 모델, base learners)들이 예측한 값을 입력 특성으로 사용하여 메타 모델이 예측을 수행합니다.
  • 다른 앙상블 기법과의 차이점은, 단순한 평균이나 투표 방식이 아니라 메타 모델을 학습한다는 점입니다.

스태킹 모델은 두 종류의 모델이 필요하다

  1. 개별 모델
  1. 이 개별 모델의 예측 데이터를 학습 데이터로 만들어서 학습하는 최종 메타 모델

스태킹 모델의 핵심은 여러 개별 모델의 예측 데이터를 각각 스태킹 형태로 결합해 최종 메타 모델의 학습용 피처 데이터 세트와 테스트용 피처 데이터 세트를 만드는 것이다.

  • 코드
    from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.ensemble import RandomForestClassifier, GradientBoostingClassifier, StackingClassifier
from sklearn.metrics import accuracy_score

# 데이터셋 준비
X, y = make_classification(n_samples=1000, n_features=20, random_state=42)

# 데이터셋 분리
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 개별 모델 정의
base_model1 = ('lr', LogisticRegression())
base_model2 = ('rf', RandomForestClassifier(n_estimators=50, random_state=42))
base_model3 = ('gb', GradientBoostingClassifier(n_estimators=50, random_state=42))

# 메타 모델 정의
meta_model = LogisticRegression()

# StackingClassifier 생성
stacking_model = StackingClassifier(
    estimators=[base_model1, base_model2, base_model3],
    final_estimator=meta_model
)

# 모델 학습
stacking_model.fit(X_train, y_train)

# 모델 예측
y_pred_stacking = stacking_model.predict(X_test)

# 모델 평가
accuracy_stacking = accuracy_score(y_test, y_pred_stacking)
print("Stacking Accuracy: ", accuracy_stacking)
