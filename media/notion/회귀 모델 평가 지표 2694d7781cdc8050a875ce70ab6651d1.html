회귀 모델 평가 지표

회귀

  • MAE (Mean Absolute Error) - 평균 절대 오차
    • 정의: MAE는 예측값과 실제값 사이의 차이의 절대값의 평균을 구하는 지표입니다. 각 오차의 부호를 무시하고 오차 크기만을 고려하여 평균화합니다.
      • nnn﻿ : 데이터 포인트의 개수
      • yiy_iyi​﻿ : 실제 관측값
      • y^i\hat{y}_iy^​i​﻿ : 예측값
    • 특징:
      • 모든 오차를 동일하게 취급합니다. 즉, 각 오차의 크기만을 고려하므로 이상치(outliers)에 덜 민감합니다.
      • 직관적이며, 예측값과 실제값 사이의 평균적인 차이를 이해하는 데 유용합니다.
    • 장점: 계산이 간단하고 해석이 쉬우며, 오차의 절대적인 크기를 보여줍니다.
    • 단점: MAE는 미분 가능하지 않은 절대값 함수를 사용하므로, 일부 최적화 기법에서는 사용이 어려울 수 있습니다.
  • MSE (Mean Squared Error) - 평균 제곱 오차
    • 정의: MSE는 예측값과 실제값 사이의 차이를 제곱한 후, 그 값의 평균을 구하는 지표입니다. 오차를 제곱하기 때문에 큰 오차에 더 큰 패널티를 부여합니다.
    • 특징:
      • 오차를 제곱하므로, 큰 오차에 대해 더 큰 패널티를 줍니다. 즉, 이상치에 민감합니다.
      • 제곱이기 때문에 오차의 크기에 큰 가중치를 두고, 모델이 큰 오차를 줄이도록 유도합니다.
    • 장점: 미분 가능하기 때문에 경사 하강법과 같은 최적화 알고리즘에 적합합니다.
    • 단점: 오차의 단위가 제곱되어 실제 데이터의 단위와 맞지 않기 때문에 해석이 직관적이지 않을 수 있습니다. 또한, 이상치에 민감하여 데이터에 노이즈가 많을 경우 모델의 성능 평가에 왜곡이 생길 수 있습니다.
  • RMSE (Root Mean Squared Error) - 평균 제곱근 오차
    • 정의: RMSE는 MSE의 제곱근을 취한 값으로, 실제 오차의 크기를 측정하는 데 더 직관적입니다.
    • 특징:
      • MSE의 제곱근이기 때문에 오차의 단위가 원래 데이터의 단위와 같아져 해석이 직관적입니다.
      • MSE와 마찬가지로 오차를 제곱한 후 평균을 구하므로, 이상치에 민감합니다.
    • 장점: MSE와 달리 데이터의 단위와 일치하기 때문에 오차의 크기를 이해하는 데 더 편리합니다.
    • 단점: MSE처럼 이상치에 민감하며, 오차의 분포에 따라 값이 왜곡될 수 있습니다.
  • MSLE (Mean Squared Logarithmic Error) - 평균 제곱 로그 오차
    • 정의: MSLE는 예측값과 실제값 사이의 로그 차이의 제곱을 평균한 지표입니다. 오차를 측정할 때 직접적인 차이를 구하지 않고, 로그 변환을 통해 상대적인 차이를 계산합니다.
      • nnn﻿ : 데이터 포인트의 개수
      • yiy_iyi​﻿ : 실제 관측값
      • yi^ \hat{y_i}yi​^​﻿ : 예측값
    • 특징:
      • 실제값과 예측값이 비슷할수록 MSLE 값이 작아집니다.
      • 로그 변환을 사용하기 때문에 값이 작을수록 오차에 민감하고, 값이 클수록 오차에 덜 민감합니다. 즉, 큰 값의 차이를 축소하고 작은 값의 차이에 더 집중하는 특성을 가집니다.
      • 값의 스케일이 큰 경우(예: 수백에서 수백만에 이르는 값) 상대적인 오류를 강조하고자 할 때 유용합니다.
    • 장점: 값이 작을 때 오차를 더 강조하므로, 실제값과 예측값이 큰 차이가 날 때도 모델 성능을 균형 있게 평가할 수 있습니다.
    • 단점: 값이 음수일 경우 로그를 취할 수 없으므로, MSLE는 양수 데이터에만 사용할 수 있습니다.
  • RMSLE (Root Mean Squared Logarithmic Error) - 평균 제곱근 로그 오차
    • 정의: RMSLE는 MSLE의 제곱근을 취한 값으로, 실제값과 예측값 사이의 상대적 차이를 측정하는 데 사용됩니다.

    • 특징:
      • MSLE의 단위를 원래 데이터의 단위와 일치시켜 해석을 더 직관적으로 할 수 있습니다.
      • 큰 값의 오차보다는 작은 값의 오차를 강조하므로, 예측값이 실제값보다 많이 낮게 나올 때 패널티를 적게 주는 특성이 있습니다.
      • 음수값이 없는 데이터에서 사용해야 합니다.
    • 장점: 상대적인 오차를 측정하기 때문에 값이 크게 변하는 경우에도 오차를 안정적으로 평가할 수 있습니다.
    • 단점: 실제값이 0에 매우 가까울 때 계산이 복잡해지거나 왜곡될 수 있으므로, 이 경우 처리에 주의해야 합니다.
  • R2R^2R2﻿ (Coefficient of Determination) - 결정계수
    • 정의: R2R^2R2﻿는 모델이 종속 변수의 변동을 얼마나 잘 설명하는지를 나타내는 지표로, 모델의 적합도를 평가하는 데 사용됩니다.
      • yiy_iyi​﻿: 실제 관측값
      • yi^\hat{y_i}yi​^​﻿: 예측값
      • yˉ\bar{y}yˉ​﻿: 실제 관측값의 평균
      • nnn﻿ : 데이터 포인트의 개수
    • 특징:
      • R2R^2R2﻿의 값은 0에서 1 사이이며, 1에 가까울수록 모델이 데이터를 잘 설명한다는 것을 의미합니다.
      • R2R^2R2﻿ = 1이면 예측값이 실제값과 완벽하게 일치하는 경우입니다.
      • R2R^2R2﻿ = 이면 모델이 종속 변수의 변동을 전혀 설명하지 못하고, 단순히 평균값을 사용한 것과 같습니다.
      • 음수의 R2R^2R2﻿도 나올 수 있는데, 이는 모델이 데이터의 변동을 평균보다도 더 못 설명한다는 의미입니다.
    • 장점: 모델이 데이터를 얼마나 잘 설명하는지 한눈에 파악할 수 있어 회귀 모델의 성능을 평가하는 데 매우 유용합니다.
    • 단점: 독립 변수의 개수가 늘어날수록 R2R^2R2﻿ 값이 무조건 증가하는 경향이 있으므로, 조정된 결정계수(Adjusted R2R^2R2﻿)를 사용해 모델의 과적합을 방지할 수 있습니다.
  • 코드
    from sklearn.metrics import mean_absolute_error, mean_squared_error, r2_score
import numpy as np

# 예시 실제값(y_true)과 예측값(y_pred)
y_true = np.array([3, -0.5, 2, 7])
y_pred = np.array([2.5, 0.0, 2, 8])

# MAE (Mean Absolute Error)
mae = mean_absolute_error(y_true, y_pred)
print(f"MAE: {mae}")

# MSE (Mean Squared Error)
mse = mean_squared_error(y_true, y_pred)
print(f"MSE: {mse}")

# RMSE (Root Mean Squared Error)
rmse = np.sqrt(mse)
print(f"RMSE: {rmse}")

# MSLE (Mean Squared Logarithmic Error)
msle = mean_squared_error(np.log1p(y_true), np.log1p(y_pred))
print(f"MSLE: {msle}")

# RMSLE (Root Mean Squared Logarithmic Error)
rmsle = np.sqrt(msle)
print(f"RMSLE: {rmsle}")

# R^2 (Coefficient of Determination)
r2 = r2_score(y_true, y_pred)
print(f"R^2: {r2}")
