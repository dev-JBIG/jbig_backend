# 데이터셋 준비
from pycaret.datasets import get_data
data = get_data('diabetes')

일반적인 WorkFlow 5단계

Setup ➡️ Compare Models ➡️ Analyze Model ➡️ Prediction ➡️ Save Model

1. Setup

이 함수는 훈련 환경을 초기화하고 변환 파이프라인을 생성합니다. PyCaret에서 다른 함수를 실행하기 전에 Setup 함수를 호출해야 합니다. 이 함수에는 data와 target이라는 두 가지 필수 파라미터만 있습니다. 다른 모든 파라미터는 선택 사항입니다.

# setup 함수 호출하기
from pycaret.classification import *
s = setup(data, target = 'Class variable', session_id = 123)

2. Compare Models

이 함수는 교차 검증을 사용하여 모델 라이브러리에서 사용 가능한 모든 estimators의 성능을 훈련하고 평가합니다. 이 함수의 출력값은 교차 검증 평균 점수가 포함된 점수표입니다.

# 여러 모델 비교하기
# 이를 통해 어떤 모델이 해당 데이터에서 
# 우수한 성능을 보이는지 파악할 수 있습니다.
best = compare_models()

3. Analyze Model

plot_model 함수를 사용하여 테스트 세트에서 학습된 모델의 성능을 분석할 수 있습니다. 경우에 따라 모델을 다시 학습시켜야 할 수도 있습니다.

# confusion matrix(오차행렬) 그리기
plot_model(best, plot = 'confusion_matrix')
# feature importance 그리기
# feature importance란 
# 학습하는데 가장 영향을 많이 미친 피쳐부터 나열한 것
plot_model(best, plot = 'feature')

4. Prediction

predict_model 함수는 데이터 프레임의 새 열로 prediction_label 및 prediction_score(예측된 클래스의 확률)를 반환합니다. data가 None(default)인 경우, Setup 함수 중에 생성된 test set가 점수 측정에 사용됩니다.

# 테스트 데이터에 예측하기
holdout_pred = predict_model(best)
# 새 데이터에 예측하기
predictions = predict_model(best, data = new_data)
predictions.head()

5. Save Model

마지막으로 PyCaret의 save_model 함수를 사용하여 나중에 사용할 수 있도록 전체 파이프라인을 저장할 수 있습니다.

# 파이프라인 저장하기
save_model(best, 'my_first_pipeline')
# 파이프라인 불러오기
loaded_best_pipeline = load_model('my_first_pipeline')
loaded_best_pipeline
