활성화 함수 (Activation Function)

개요

활성화 함수는 신경망의 각 뉴런에서 출력값을 결정하는 핵심 요소입니다. 활성화 함수가 없다면 신경망은 단순히 선형 계산만 수행하게 되어, 깊은 층을 쌓아도 복잡한 비선형 문제를 해결할 수 없습니다. 비선형 활성화 함수를 통해 신경망은 이미지 인식, 음성 인식, 자연어 처리와 같은 복잡한 문제를 학습할 수 있습니다.

활성화 함수가 쓰이는 곳

  • 은닉층과 출력층: 입력층에서 받은 데이터를 은닉층에서 비선형으로 변환해 학습할 수 있도록 돕습니다. 특히 다층 퍼셉트론(MLP)과 컨볼루션 신경망(CNN) 등 대부분의 신경망에서 활성화 함수는 필수적입니다.
  • 출력층: 이진 분류, 다중 클래스 분류, 회귀 문제에 따라 시그모이드, 소프트맥스 등이 사용됩니다.

활성화 함수는 신경망이 복잡한 패턴을 학습할 수 있게 하는 기본 요소로, 각 함수의 특성을 이해하고 적용할 수 있어야 문제에 맞는 성능을 이끌어낼 수 있습니다. 활성화 함수를 잘 선택하면 신경망의 성능과 학습 효율이 크게 향상되기 때문에, 이를 잘 이해하는 것은 딥러닝 학습에 필수입니다.

활성화 함수의 종류

여러 종류의 활성화 함수가 있으며, 각 함수는 특성이 다르기 때문에 문제에 따라 적합한 함수를 선택할 수 있습니다. 주요 활성화 함수는 다음과 같습니다.

그냥 아, 이런게 있구나, 그래프가 이렇게 형성되는구나 정도만 파악하셔도 됩니다.

1. 시그모이드 함수 (Sigmoid Function)

  • 정의:
  • 출력 범위: 0에서 1 사이
  • 특징: 입력 값을 0과 1 사이로 압축합니다. 주로 이진 분류 문제에서 출력층 활성화 함수로 사용됩니다.
  • 단점: 큰 양수나 음수에서는 그래디언트가 매우 작아지는 기울기 소실(Vanishing Gradient) 문제가 발생할 수 있습니다.
# Sigmoid 활성화 함수
sigmoid = nn.Sigmoid()
output = sigmoid(x)

2. 하이퍼볼릭 탄젠트 함수 (Tanh Function)

  • 정의:
  • 출력 범위: -1에서 1 사이
  • 특징: 시그모이드 함수와 비슷하지만, 출력이 -1에서 1까지 분포하여 중심이 0입니다. 신경망이 더 잘 수렴하도록 도울 수 있습니다.
  • 단점: 시그모이드와 마찬가지로 기울기 소실 문제가 발생할 수 있습니다.
# Tanh 활성화 함수
tanh = nn.Tanh()
output = tanh(x)

3. 렐루 함수 (ReLU, Rectified Linear Unit)

  • 정의:
  • 출력 범위: 0 이상
  • 특징: 음수는 0으로, 양수는 그대로 출력하여 연산이 간단하고, 기울기 소실 문제가 덜 발생합니다. 현재 신경망에서 가장 널리 사용되는 활성화 함수 중 하나입니다.
  • 단점: 음수 영역에서 값이 0으로 고정되는 죽은 ReLU(Dying ReLU) 문제 발생 가능.
# ReLU 활성화 함수
relu = nn.ReLU()
output = relu(x)

4. 리키 렐루 함수 (Leaky ReLU)

  • 정의:
  • 출력 범위: 음수에서는 0.01x0.01x0.01x﻿, 양수는 그대로 출력
  • 특징: ReLU의 변형으로, 음수 영역에서 작은 기울기를 주어 죽은 ReLU 문제를 완화합니다.
  • 장점: 모든 입력에 대해 작은 기울기를 유지하여 학습 속도가 개선될 수 있습니다.
# Leaky ReLU 활성화 함수
leaky_relu = nn.LeakyReLU(negative_slope=0.01)
output = leaky_relu(x)

5. ELU 함수 (Exponential Linear Unit)

  • 정의:
  • 출력 범위: 음수에서의 출력은 α\alphaα﻿에 의해 조정됩니다.

  • 특징: Leaky ReLU와 비슷하게 음수 값에서도 작은 기울기를 가지며, ReLU보다 수렴 속도가 빠릅니다.
  • 단점: 계산량이 약간 더 많아집니다.
# ELU 활성화 함수 (alpha=1.0)
elu = nn.ELU(alpha=1.0)
output = elu(x)

6. 소프트맥스 함수 (Softmax Function)

  • 정의:
  • 출력 범위: 0에서 1 사이, 모든 출력의 합은 1
  • 특징: 주로 다중 클래스 분류 문제의 출력층에서 사용되며, 각 클래스에 대한 확률값을 제공합니다.
  • 장점: 여러 클래스 중 확률이 가장 높은 클래스를 예측할 수 있게 합니다.
# Softmax 활성화 함수
softmax = nn.Softmax(dim=0)
output = softmax(x)
