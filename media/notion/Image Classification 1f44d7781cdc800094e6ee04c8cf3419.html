Image Classification

Image Classification Definition

인풋 이미지에 대해 라벨링을 수행하는 작업으로 컴퓨터 비전의 핵심 과제이다.

The Problem: Semantic Gap

인간의 시각 추론과 달리 컴퓨터는 이미지를 수를 포함한 행렬로 인식한다. 따라서 인간은 같은 종류(라벨)의 다른 이미지라도 시각 추론을 바탕으로 동일한 라벨링 작업을 수행할 수 있는 반면에, 컴퓨터가 인식하는 이미지는 같은 종류(라벨)의 이미지라도 다른 구성, 다른 환경 등에서 나타난 데이터라면 완전히 다른 숫자의 집합으로 인식하게 된다. 따라서 컴퓨터 비전에서 Image Classification 수행 시 고려해야할 사항은 다음과 같다.

  • View Point Variation - 카메라의 위치 변화
  • Illumination Condition - 조명 조건
  • DeFormation - 형태 변화
  • Occlusion - 가려짐
  • BackGround Clutter - 배경과 비슷함
  • Intra - class Variation - 같은 클래스에서의 차이

First classifier : Nearest Neighbor ( 최소 근접 이웃 알고리즘)

Nearest Neighbor Algorithms 다음과 같이 작동한다.

  • Train : 아무것도 하지 않고 Train data를 전부 암기한다.
  • Predict : Train data 에서 현재 분류를 시도하고자 하는 이미지와 가장 유사한 이미지의 라벨을 통해 예측을 시도한다.

L1 Distance : Manhattan

Predict 단계에서 이미지와 가장 유사한 이미지를 이용한다고 하였는데, 이미지의 유사도를 측정하는 방법이다.

위와 같은 방법으로 픽셀간의 차이의 합를 계산해서 총합으로 유사도를 측정한다.

import numpy as np

class NearestNeighbor:
	def __init__(self):
		pass
	
	def train(self, X, y):
		# train 데이터에 대해 모든 라벨을 암기 한다.
		self.Xtr = X
		self.ytr = y
	
	def predict(self,X):
		num_test = X.shape[0]
		y_pred = np.zeros(num_test, dtype = self.ytr.dtype)
		
		for i in range(num_test):
			# 이미지간 유사도를 L1 distance를 이용하여 측정하고 훈련데이터에서 가장 작은
			# 값을 가지는 이미지의 라벨을 예측 라벨로 사용하는 것을 볼 수 있다.
			distances = np.sum(np.abs(self.Xtr - X[i,:]),axis = 1)
			min_index = np.argmin(distances)
			y_pred[i] = self.ytr[min_index]
		
		return y_pred

위 코드를 보았을 때 다음과 같은 특징이 있다. 훈련 시간복잡도 O(1), 예측 시간복잡도 O(N)

이것은 매우 좋지 않은 경우이다. 왜냐하면 보통 모델을 사용할 때, Prediction 에는 빠르기를 원하고 Training 할 때 느린 것을 감안하기 때문이다.
