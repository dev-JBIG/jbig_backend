앙상블

  • 개념

    앙상블의 개념

    앙상블 방법의 기본 아이디어는 여러 모델의 예측을 결합하여 강력한 모델을 구축하는 것입니다.

    특정 약점이 있을 수 있는 단일 모델에 의존하는 대신 앙상블은 다양한 모델을 사용하여 더 나은 정확성과 일반화 성능을 보여줍니다.

    모델 간의 집단지성이라고 보면 됨

  • 필요성

    앙상블을 하는 이유

    정확도 향상: 앙상블은 여러 모델을 결합하여 개별 모델보다 더 높은 정확도를 달성할 수 있습니다.

    과적합(Overfitting) 감소: 앙상블은 개별 모델의 편향을 평균화하여 과적합을 줄이는 데 도움이 될 수 있습니다.

    일반화(Geralization) : 다양한 데이터 세트에서 더욱 안정적이고 신뢰할 수 있는 성능을 제공합니다.

    편향-분산 트레이드오프: 앙상블은 편향(지나치게 단순한 모델로 인한 오류)과 분산(지나치게 복잡한 모델로 인한 오류) 간의 트레이드 오프 균형을 맞출 수 있습니다.

보팅

1) 보팅(Voting)

보팅(Voting)은 일반적으로 같은 데이터로 학습시킨 서로 다른 알고리즘 기반의 모델의  결과를 결합합니다.

보팅은 크게 하드 보팅(hard voting)과 소프트 보팅(soft voting)으로 나뉩니다.

1-1) 하드 보팅(hard voting)

우리가 흔히 아는 다수결 투표와 같은 개념이다. 각 모델의 결과를 취합하여 가장 많이 예측된 값으로 최종 예측 값을 선정하는 것이다.

  • 코드
    from sklearn.linear_model import LogisticRegression
from sklearn.ensemble import RandomForestClassifier, GradientBoostingClassifier
from sklearn.ensemble import VotingClassifier
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 데이터셋 준비
X, y = make_classification(n_samples=1000, n_features=20, random_state=42)

# 데이터셋 분리
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 모델 정의
model1 = LogisticRegression()
model2 = RandomForestClassifier(n_estimators=50, random_state=42)
model3 = GradientBoostingClassifier(n_estimators=50, random_state=42)

# 하드 보팅 분류기 생성
voting_model_hard = VotingClassifier(estimators=[
    ('lr', model1), ('rf', model2), ('gb', model3)],
    voting='hard')

# 모델 학습
voting_model_hard.fit(X_train, y_train)

# 모델 예측
y_pred_hard = voting_model_hard.predict(X_test)

# 모델 평가
accuracy_hard = accuracy_score(y_test, y_pred_hard)
print("Hard Voting Accuracy: ", accuracy_hard)

1-2) 소프트 보팅(soft voting)

예측 클래스가 아닌 클래스 당 예측 확률(predict_proba의 리턴값)을 모든 모델로부터 받아와서, 해당 클래스 당 예측 확률 값을 평균 내어 가장 높은 확률을 갖는 클래스로 최종 예측을 하는 것이다.

  • 코드
    from sklearn.linear_model import LogisticRegression
from sklearn.ensemble import RandomForestClassifier, GradientBoostingClassifier
from sklearn.ensemble import VotingClassifier
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 데이터셋 준비
X, y = make_classification(n_samples=1000, n_features=20, random_state=42)

# 데이터셋 분리
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 모델 정의
model1 = LogisticRegression()
model2 = RandomForestClassifier(n_estimators=50, random_state=42)
model3 = GradientBoostingClassifier(n_estimators=50, random_state=42)

# 소프트 보팅 분류기 생성
voting_model_soft = VotingClassifier(estimators=[
    ('lr', model1), ('rf', model2), ('gb', model3)],
    voting='soft')

# 모델 학습
voting_model_soft.fit(X_train, y_train)

# 모델 예측
y_pred_soft = voting_model_soft.predict(X_test)

# 모델 평가
accuracy_soft = accuracy_score(y_test, y_pred_soft)
print("Soft Voting Accuracy: ", accuracy_soft)

배깅

배깅(Bagging)

배깅(Bagging)은 보팅(Voting)과 다르게, 단일 알고리즘 기반의 모델을 다른 데이터를 기반으로 학습시켜 결과를 결합한다. 

Bootstrap Aggregating의 약자로서 불리는 것이다. 부트스트랩은 통계학에서 사용하는 용어로, random sampling을 적용하는 방법이다. 말그대로 Bootstrapping을 통해 추출한 데이터로 각 모델을 학습시키고 이를 aggregating, 합계하여 최종 결과를 도출해낸다.

  • 코드
    from sklearn.ensemble import BaggingClassifier
from sklearn.tree import DecisionTreeClassifier
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 데이터셋 준비
X, y = make_classification(n_samples=1000, n_features=20, random_state=42)

# 데이터셋 분리
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 모델 정의
base_model = DecisionTreeClassifier()

# 배깅 분류기 생성
bagging_model = BaggingClassifier(base_estimator=base_model, n_estimators=50, random_state=42)

# 모델 학습
bagging_model.fit(X_train, y_train)

# 모델 예측
y_pred_bagging = bagging_model.predict(X_test)

# 모델 평가
accuracy_bagging = accuracy_score(y_test, y_pred_bagging)
print("Bagging Accuracy: ", accuracy_bagging)
