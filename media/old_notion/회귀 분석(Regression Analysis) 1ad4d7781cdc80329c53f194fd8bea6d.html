회귀 분석(Regression Analysis)

선형회귀

  • 단순 선형 회귀(Simple Linaer Regression)
    • 1. 개념

      Linear Regression은 머신 러닝에서 가장 기본적인 예측 모델 중 하나로, yyy﻿(종속 변수)와 하나 이상의 xxx﻿(독립변수) 간의 선형 관계를 모델링하는 기법입니다.

      이 방법은 주어진 데이터에 가장 잘 맞는 직선(또는 초평면)을 찾음으로써 종속 변수를 예측합니다.

      즉, 선형회귀란 데이터를 가장 잘 대변하는 최적의 선을 찾는 과정입니다.

      독립 변수 xxx﻿가 하나라면 단순 선형 회귀(Simple Linaer Regression)

      독립 변수 xxx﻿가 2개 이상이면 다중 선형 회귀(Multiple Linear Regression)라고 한다.

      단순 선형 회귀 : y=β1x+β0+ϵy=β_1x+β_0+\epsilony=β1​x+β0​+ϵ﻿

      다중 선형 회귀 : y=β0+β1x1+β2x2+...wnxn+ϵy=β_0+β_1x_1+β_2x_2+...w_nx_n+\epsilony=β0​+β1​x1​+β2​x2​+...wn​xn​+ϵ﻿

      예를 들어, 광고 비용에 따라 판매량을 예측하거나, 집의 크기와 방의 수를 통해 집의 가격을 예측하는 데 사용될 수 있습니다.

    • 2. 수학적 표현
      • 단순 선형 회귀(Simple Linear Regression) : 독립 변수가 하나일 때 사용됩니다. 수학적으로 다음과 같이 표현됩니다 :
        • y=β0+β1x+ϵy = \beta_0+\beta_1x+\epsilony=β0​+β1​x+ϵ﻿
        • yyy﻿ : 종속 변수
        • xxx﻿ : 독립 변수
        • β0\beta_0β0​﻿ : y-절편 (회귀선이 y축과 만나는 점)
        • β1\beta_1β1​﻿ : 기울기 (독립 변수와 종속 변수 간의 관계를 나타냄)
        • ε\varepsilonε﻿ : 오차(에러) 항

          지난번에 hyperparameter vs parameter 에서 다뤘던 것 처럼 모델은 parameter들을 스스로 조정해나가면서 학습(최적화)을 해나갑니다.

          여기서는 β0,β1\beta_0, \beta_1β0​,β1​﻿이 parameter가 되겠군요.

          그렇다면 머신러닝은 최적의 선을 찾기 위해 어떻게 parameter(기울기와 yyy﻿절편값)를 찾아 나갈까요?

          간단히 설명하자면, 모델은 Error를 최소화 하기 위한 방법으로 점차적으로 학습해나간다는것입니다.

          그 방법을 알아보기 이전에, Error를 먼저 정의하도록 하겠습니다.

    • 3. Loss Fuction(손실 함수)

      선형 회귀에서 일반적으로 사용되는 손실 함수는 평균 제곱 오차(MSE, Mean Squared Error)입니다.

      평균 제곱 오차 (MSE)

      • 정의: MSE는 예측값과 실제값 간의 차이(오차)를 제곱한 후, 그 값의 평균을 구하는 것입니다.

        MSE=1n∑i=1n(yi−yi^)2MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y_i})^2 MSE=n1​∑i=1n​(yi​−yi​^​)2﻿

        • nnn﻿ : 데이터 포인트의 총 개수
        • yiy_iyi​﻿ : 실제 관측값
        • yi^\hat{y_i}yi​^​﻿ 모델이 예측한 값
      • 특징:
        • 오차를 제곱하므로, 모든 오차가 양수로 변환되어 더해집니다. 따라서 큰 오차에 더 큰 패널티를 부여합니다.
        • 손실 함수가 최소가 되는 지점에서 모델이 데이터에 가장 잘 맞는다고 판단합니다.
      • 다른 손실함수들

      이제 Error를 정의했습니다. 이렇게 정의한 Error는 어떤 방식으로 줄어들게 학습이 되는 것이며 parameter들은 어떻게 조정되는 것 일까요? 가장 기본적인 방법은 경사하강법입니다.

    • 4. 최적화 : 경사 하강법(Gradient Descent) :

      경사 하강법은 손실 함수를 반복적으로 최소화하는 최적화 알고리즘으로, 가장 일반적인 방법입니다. Error가 최소값이 되는 최적해(Gobal Minimum)을 찾는 것이 목표입니다.

      이 방법은 다음과 같은 단계로 이루어집니다.

      단계별 과정:

      1. 초기화: 모델의 파라미터(가중치와 절편)를 임의의 값으로 초기화합니다. 예를 들어, β0\beta_0β0​﻿와 β1\beta_1β1​﻿을 임의로 설정합니다.
      1. 예측 및 손실 계산:
        • 모델을 사용해 예측값 yi^\hat{y_i}yi​^​﻿를 계산하고, 각 관측치 yiy_iyi​﻿와 비교하여 손실 함수를 계산합니다. 이때 손실 함수로 주로 평균 제곱 오차를 사용합니다:

          MSE=1n∑i=1n(yi−yi^)2MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y_i})^2 MSE=n1​∑i=1n​(yi​−yi​^​)2﻿

      1. 경사(Gradient) 계산:
        • 손실 함수를 현재 파라미터에 대해 미분하여 각 파라미터에 대한 기울기(gradient)를 구합니다. 이때 파라미터가 단 두 개라면 다음과 같이 표현됩니다:

          ∂MSE∂β0=−2n∑i=1n(yi−yi^)\frac{\partial \text{MSE}}{\partial \beta_0} = -\frac{2}{n} \sum_{i=1}^{n} (y_i - \hat{y_i}) ∂β0​∂MSE​=−n2​∑i=1n​(yi​−yi​^​)﻿

          ∂MSE∂β1=−2n∑i=1n(yi−yi^)xi\frac{\partial \text{MSE}}{\partial \beta_1} = -\frac{2}{n} \sum_{i=1}^{n} (y_i - \hat{y_i}) x_i∂β1​∂MSE​=−n2​∑i=1n​(yi​−yi​^​)xi​﻿

      1. 파라미터 업데이트:
        • 각 파라미터를 기울기에 따라 업데이트합니다. 기울기는 손실 함수가 가장 크게 변하는 방향을 나타내므로, 파라미터를 이 기울기의 반대 방향으로 조정하여 손실을 줄여나갑니다. 이때 학습률(learning rate) α\alphaα﻿를 곱해 파라미터 업데이트의 크기를 조절합니다.
          • 학습률(learning rate)이란?

            학습률(Learning Rate)은 경사 하강법에서 모델의 가중치를 업데이트할 때 사용되는 단계(Step)의 크기를 결정하는 하이퍼파라미터입니다.

            • 크기가 작을수록 : 최적해(Global Minimum)에 수렴하는 데 시간이 오래 걸리지만, 안정적으로 수렴(Convergence)할 가능성이 높습니다.
            • 크기가 클수록: 수렴 속도가 빨라질 수 있지만, 최적해(Global Minimum)를 지나치거나 발산(Divergence)하여 학습이 실패할 위험이 있습니다.

              학습률은 경사 하강법의 수렴 속도와 안정성을 조절하는 중요한 요소입니다.

          β0:=β0−α∂MSE∂β0 \beta_0 := \beta_0 - \alpha \frac{\partial \text{MSE}}{\partial \beta_0} β0​:=β0​−α∂β0​∂MSE​ ﻿

          β1:=β1−α∂MSE∂β1 \beta_1 := \beta_1 - \alpha \frac{\partial \text{MSE}}{\partial \beta_1} β1​:=β1​−α∂β1​∂MSE​ ﻿

      1. 반복 :
        • 이 과정을 손실 함수의 값이 충분히 작아지거나 기울기가 0에 가까워질 때까지 반복합니다.

          이 사진은 위 살명과 순서가 1:1로 매칭되지는 않지만 개념적인 흐름에서 유사합니다.

          여기서 X축인 Weight(가중치)가 parameter입니다.

          Y축은 손실함수(Loss Function)입니다.

          • 영상

      특징 :

      • 데이터의 규모에 관계없이 적용할 수 있으며, 학습률(learning rate)을 잘 조절하면 전역 최적값(Global minimum)에 수렴할 수 있습니다.
      • 단점 : 학습률이 너무 크면 수렴하지 않고 발산할 수 있고, 너무 작으면 수렴하는 데 시간이 오래 걸릴 수 있습니다.
    • error(오차)

      Error (오차):

      • 정의: 오차는 모델이 전체 모집단에 대해 예측한 값과 실제 값 간의 차이를 의미합니다. 이는 예측 모델의 이론적인 오류를 나타내며, 주어진 모집단 전체에서의 불확실성을 포함합니다.
      • 표현:
        • 실제 값과 모델의 참된 예측 값(모수적 관계) 간의 차이를 나타냅니다.
        • 수학적으로, 모집단에서 iii﻿번째 데이터의 오차는 다음과 같이 표현됩니다:

          εi=yi−f(xi)ε_i=y_i−f(x_i)εi​=yi​−f(xi​)﻿

          • yiy_iyi​﻿: 실제 값
          • f(xi)f(x_i)f(xi​)﻿: 모집단에 대한 모델의 참된 예측 값
      • 특징:
        • 오차는 모집단 전체에 대한 참된 관계를 가정하며, 관측 데이터의 부분집합(표본)에 한정되지 않습니다.
        • 선형 회귀에서 우리가 알 수 없는 모델의 불확실성을 포함하며, 실제로는 측정할 수 없는 개념입니다.
        • 기본적으로 회귀직선에 이러한 오차항(error term) ϵ\epsilonϵ﻿이 더해지는 이유는 다음과 같습니다.
        1. 모든 데이터가 관측되었다는 보장이 없고, 관측된 데이터 역시 잡음(noise)가 존재할 경우를 반영한 것입니다.
        1. ϵ\epsilonϵ﻿을 추가함으로써, 모델이 약간의 선형성을 벗어나도 됨을 허용합니다.
        1. ϵ\epsilonϵ﻿이 정규분포라고 가정함으로써, 다양한 통계적 기법이 사용 가능합니다.

      요약

      • Error는 전체 모집단에 대한 모델의 이론적 오차로, 알 수 없는 참된 모델의 불확실성을 포함합니다. 실제로 측정하거나 계산할 수 없는 이론적 개념입니다.
  • 다중 선형 회귀(Multiple Linear Regression)

    다중 선형 회귀(Multiple Linear Regression)는 하나의 종속 변수(예측하려는 값, yyy﻿)와 두 개 이상의 독립 변수(설명 변수, xxx﻿) 간의 선형 관계를 모델링하는 회귀 기법입니다. 이는 단순 선형 회귀에서 하나의 독립 변수만 사용하는 것과 달리, 여러 독립 변수를 사용하여 종속 변수에 대한 더 정확한 예측을 수행할 수 있습니다.

    단순선형회귀의 확장된 버전으로 이해하시면 됩니다. 설명변수( xxx﻿)가 1개에서 여러개(nnn﻿개)로 증가합니다. 수식으로 표현하면 다음과 같습니다.

    y=β0+β1x1+β2x2+⋯+βnxn+εy = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \dots + \beta_n x_n + \varepsilon y=β0​+β1​x1​+β2​x2​+⋯+βn​xn​+ε﻿

    • yyy﻿ : 종속 변수(예측하려는 값)
    • x1,x2,…,xnx_1, x_2, \dots, x_nx1​,x2​,…,xn​﻿: 여러 독립 변수들
    • β0\beta_0β0​﻿: 절편 (y축과의 교점)
    • β1,β2,…,βn\beta_1, \beta_2, \dots, \beta_nβ1​,β2​,…,βn​﻿: 각 독립 변수에 대한 회귀 계수(가중치(Weight))
    • ε\varepsilonε﻿: 오차 항 (모델이 설명하지 못하는 부분)

      특징

      • 다중 선형 회귀는 종속 변수와 여러 독립 변수 간의 관계를 설명할 수 있기 때문에, 더 복잡하고 현실적인 문제에 적용할 수 있습니다.
      • 각 독립 변수의 회귀 계수 βi는 그 변수의 종속 변수에 대한 영향력을 나타냅니다. 즉, 다른 변수들이 고정되어 있을 때 해당 독립 변수가 종속 변수에 미치는 변화량을 나타냅니다.

      예시

      • 부동산 가격 예측: 집의 가격(종속 변수)을 예측할 때, 집의 크기, 방의 개수, 위치, 건축 연도 등 여러 요인(독립 변수)을 고려할 수 있습니다.
      • 마케팅 분석: 제품 판매량(종속 변수)을 예측할 때, 광고비, 가격, 계절, 경쟁사의 활동 등 여러 변수들을 독립 변수로 사용할 수 있습니다.

      다중 선형회귀에서도 단순 선형회귀와 같이 손실함수를 측정하고 최적의 가중치를 찾기 위한 방법으로 경사하강법 등 최적화 방식을 사용하는 것은 똑같습니다.

다항 회귀(Multi-Polynomial Regression)

  • 다항 회귀(Multi-Polynomial Regression)

    다항 회귀(Multi-Polynomial Regression)는 종속 변수와 독립 변수 간의 관계를 선형이 아닌 비선형 다항식(polynomial) 형태로 모델링하는 회귀 분석 방법입니다. 설명변수(xxx﻿)와 반응변수(yyy﻿)가 곡선 관계( curvelinear relationship)가 있는 경우에 사용됩니다. 이는 선형 회귀를 확장한 것으로, 데이터를 더 유연하게 모델링하여 복잡한 패턴을 표현할 수 있습니다.

    1. 다항 회귀의 정의

    다항 회귀는 단순 선형 회귀나 다중 선형 회귀와 달리, 독립 변수의 거듭제곱을 사용하여 종속 변수와 독립 변수 사이의 비선형 관계를 모델링합니다.

    일변수 다항 회귀 (Polynomial Regression)

    단일 독립 변수 xxx﻿와 종속 변수 yyy﻿간의 비선형 관계를 표현할 때는 다음과 같이 나타낼 수 있습니다 :

    y=β0+β1x+β2x2+β3x3+⋯+βnxn+εy = \beta_0 + \beta_1 x + \beta_2 x^2 + \beta_3 x^3 + \dots + \beta_n x^n + \varepsilony=β0​+β1​x+β2​x2+β3​x3+⋯+βn​xn+ε﻿

    • yyy﻿ : 종속 변수
    • xxx﻿ : 독립 변수
    • β0,β1,…,βn\beta_0, \beta_1, \dots, \beta_nβ0​,β1​,…,βn​﻿ : 회귀 계수 (모델 파라미터)
    • nnn﻿ : 다항식의 차수 (degree)
    • ε\varepsilonε﻿ : 오차 항

    다변수 다항 회귀 (Multi-Polynomial Regression)

    다중 다항 회귀에서는 여러 독립 변수가 존재하며, 각 변수의 다항식 항을 고려하여 모델을 구성합니다. 예를 들어, 두 개의 독립 변수 x1x_1x1​﻿과 x2x_2x2​﻿를 사용하는 2차 다항 회귀는 다음과 같이 표현됩니다

    y=β0+β1x1+β2x2+β3x12+β4x22+β5x1x2+εy = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \beta_3 x_1^2 + \beta_4 x_2^2 + \beta_5 x_1 x_2 + \varepsilony=β0​+β1​x1​+β2​x2​+β3​x12​+β4​x22​+β5​x1​x2​+ε﻿

    이처럼 다양한 차수의 항을 추가하여 비선형성을 표현합니다.

    2. 모델의 목적

    다항 회귀는 데이터를 설명하는 가장 적합한 곡선을 찾는 것을 목표로 합니다. 독립 변수와 종속 변수 간의 관계가 선형이 아닌 경우, 다항식 항을 사용하면 더 복잡하고 유연한 모델을 만들 수 있습니다.

    3. 특징

    • 선형 회귀의 확장: 다항 회귀는 회귀 모델이 독립 변수의 거듭제곱(다항식) 항을 포함함으로써 비선형 패턴을 포착하도록 합니다. 그러나 회귀 계수 β\betaβ﻿들이 선형적으로 결합되기 때문에 여전히 선형 회귀의 한 유형으로 간주됩니다.
    • 다항식 차수(degree) 선택: 모델의 차수 nnn﻿을 선택하는 것은 매우 중요합니다.
      • 낮은 차수: 데이터의 비선형 패턴을 제대로 설명하지 못할 수 있습니다. 이는 언더피팅(underfitting)으로 이어집니다.
      • 높은 차수: 너무 많은 굴곡을 가진 곡선을 만들어 과도하게 데이터를 설명할 수 있습니다. 이는 오버피팅(overfitting)으로 이어지며, 새로운 데이터에 대한 예측 성능이 떨어집니다.

    4. 모델 학습

    • 특성 확장: 다항 회귀를 적용하려면 먼저 독립 변수를 다항식 항으로 확장해야 합니다. 예를 들어, x1x_1x1​﻿이 독립 변수인 경우, 이를 x1,x12,x13,…,x1nx_1, x_1^2, x_1^3, \dots, x_1^nx1​,x12​,x13​,…,x1n​﻿로 확장합니다.
    • 평균 제곱 오차(MSE) 최소화: 확장된 특성을 사용해 선형 회귀와 동일하게 평균 제곱 오차(MSE)를 최소화하여 회귀 계수들을 학습합니다.

    5. 예시

    • 온도 예측: 온도 변화가 하루 중 시간의 복잡한 함수인 경우, 단순한 선형 모델로는 설명하기 어려울 수 있습니다. 다항 회귀를 사용하면 비선형적인 온도 변화를 모델링할 수 있습니다.

    6. 장단점

    • 장점: 비선형 관계를 더 잘 모델링할 수 있으므로, 독립 변수와 종속 변수 간의 복잡한 패턴을 포착할 수 있습니다.
    • 단점:
      • 차수가 높을수록 모델이 복잡해지며, 오버피팅의 위험이 커집니다.
      • 데이터의 노이즈에 민감할 수 있습니다.
      • 과도한 다항식 항을 포함할 경우, 해석이 어려워질 수 있습니다.

    7. 다항 회귀의 과적합 방지

    • 차수 제한: 모델의 차수를 적절하게 제한하여 과적합을 방지할 수 있습니다.
    • 교차 검증: 교차 검증을 통해 모델이 새로운 데이터에 대해 얼마나 잘 일반화하는지 평가하고 적절한 차수를 선택합니다.
